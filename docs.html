Add to Server

Welcome to the official Snax documentation. Below is the comprehensive list of every command available, categorized by permission groups.

Note: The default prefix is $. This can be changed per-server by an Admin.

Music & Default Utilities

$play <song/url>
Plays a track from YouTube or adds it to the queue.
$skip
Skips the current track.
$prev
Plays the previous track in the queue.
$queue
Displays the current music queue.
$nowplaying
Displays the currently playing track.
$volume <0-100>
Adjusts the playback volume.
$pause / $resume
Pauses or resumes playback.
$remove <number>
Removes a specific track from the queue.
$clear
Clears the entire music queue.
$shuffle
Shuffles the current queue.
$loop
Toggles loop mode for the current track or queue.
$autoplay
Toggles automatic playback of recommended songs.
$stop
Stops playback and clears the queue.
$join / $leave
Forces the bot to join or leave your voice channel.
$ping
Checks bot latency and API response time.
$help
Displays the interactive help menu.
$hello
Bot says hello!

VoiceExe Permissions

$mute / $unmute @user
Server mutes or unmutes a user.
$deafen / $undeafen @user
Server deafens or undeafens a user.
$dragreq @user
Sends an interactive request to pull a user to your channel.
$addme
Request to be added to a locked voice channel.

ChatExe Permissions

$purge <1-100>
Bulk deletes messages in the current text channel.
$timeout @user <minutes> <reason>
Applies a Discord timeout to a user. Reason is strictly required.

ManagerExe Permissions

$kick @user
Kicks a user. Interactive confirmation required. Cannot target equal/higher roles.
$ban / $unban @user
Bans or unbans a user. Interactive confirmation required.

AdminExe Permissions

$AdminExe / $remAdminExe @target
Assigns or revokes Admin powers (10-minute cooldown queue applies).
$ManagerExe / $remManagerExe @target
Assigns or revokes Manager powers.
$ChatExe / $remChatExe @target
Assigns or revokes ChatExe powers.
$VoiceExe / $remVoiceExe @target
Assigns or revokes VoiceExe powers.
$setNickExe / $remNickExe @target
Assigns or revokes Nickname management powers.
$assign_default / $default_remove @target
Overrides default group access.
$BypassExe / $remBypassExe @target
Grants extended spam limits to a user or role.
$SupBypass / $remSupBypass @target
Grants total immunity from the anti-spam filter.
$setprefix <newprefix>
Sets a custom prefix for the server.
$remprefix
Removes the custom prefix, reverting to the default.
