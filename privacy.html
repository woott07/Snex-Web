Privacy Policy

How we handle, secure, and protect your server and user data.

Add to Server

Effective Date: January 1, 2026

1. Data Collection

Snax only collects the necessary data required to function as an enterprise moderation and music bot. This includes Discord Server IDs, Role IDs, and User IDs strictly for the purpose of maintaining the Role-Based Access Control (RBAC) permissions.json database.

2. Message Content

Snax reads messages to detect commands (starting with the prefix) and to monitor for spam if the Anti-Spam engine is enabled. Snax does not store or log message content outside of immediate volatile memory required for the anti-spam window (10 seconds).

3. Data Sharing

We do not sell, trade, or otherwise transfer to outside parties any of your Server or User data. Your configuration files remain strictly local to the bot's hosting environment.

4. Data Removal

If you remove Snax from your server, your permissions configuration will remain in our local database to preserve settings in case you re-invite the bot. To request complete deletion of your server's config, please contact the bot owner.
