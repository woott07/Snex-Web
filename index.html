Key Pillars

Lavalink Audio Engine Snax Only

Offloads heavy decoding tasks to a high-speed Lavalink server. Supports crystal-clear voice streaming from Spotify, YouTube, SoundCloud, and Apple Music without server lag.

Granular RBAC System Musico Only

Enforces local Role-Based Access Control database permissions (VoiceExe, ChatExe, ManagerExe, AdminExe). Managers can allocate moderation rights without risking full Server Administrator overrides.

Interactive Onboarding Musico Only

Build custom verification channels with button agreement logic, private text verification chat rooms, and automated rule questionnaires with staff approve/deny panel features.

Rolling-Window Anti-Spam Shared

Smart rate limiters monitoring commands inside a rolling 10-second window. Issues clean warnings and triggers automatic 2-minute Discord user timeouts to block command floods.
